Extraindo Informações dos Dados no Power Query

Aprenda como a aba de "exibição" do Power Query pode te ajudar entender mais sobre os dados.


Quando estamos iniciando o nosso processo de ETL no Power BI na maioria das vezes teremos que fazer alguns tratamentos em nossa base de dados. Por meio do Power Query, é necessário fazer essa estruturação de dados para que futuramente o modelo de dados ganhe em performance e não tenha problemas com tipos de dados incorretos ou valores nulos.
Durante o processo de estruturação dos dados, existem diversas perguntas que devemos nos fazer para que ajudem a a planejar quais passos iremos seguir, algumas delas são:

  • Existem valores nulos?
  • As colunas estão com os tipos de dados corretos?
  • Como está a distribuição de dados em cada coluna?
  • Existe alguma coluna que não é relevante para a nossa análise? Pode ser removida?

No Power BI, existem diversas maneiras para solucionar estes problemas. Neste artigo eu gostaria de te mostrar uma ferramenta incrível do Power Query que te ajuda a ter um entendimento prévio em relação aos dados e que pode ser um meio para identificar possíveis erros.

Let´s go!

Neste exemplo, irei utilizar o "Hello, world!" dos projetos de Ciência de Dados que é a base de dados do naufrágio do Titanic que está disponível para ser baixada no site do Kaggle.
Na aba exibição no Power Query, existe um campo chamado de visualização de dados:


Através dessa ferramenta podemos extrair características dos nossos dados apenas marcando as opções acima.

Qualidade da Coluna

Selecionando esta opção o resultado será o percentual de valores válidos, erros e valores nulos em cada uma das colunas, segue abaixo um exemplo:


Se posicionarmos o mouse em cima da legenda, irá aparecer uma caixinha mostrando o percentual de cada um desses valores. Clicando no sinal de reticências (três pontinhos) existem diversas opções de tratamento para possíveis erros como remover vazio, remover duplicatas, substituir erros, entre outras opções.

Distribuição de Dados nas Colunas

Se selecionarmos o campo "Distribuição de colunas", este será o resultado:


Na parte superior, embaixo do nome das colunas irá aparecer a distribuição dos dados de cada coluna. Se nós posicionarmos o mouse em cima de um dos gráficos, ele nos mostra a quantidade valores distintos e exclusivos pertencentes a coluna selecionada.
Igual a opção Qualidade da Coluna, aqui também podemos posicionar o mouse em cima das visualizações e fazer tratamentos diretamente clicando no sinal de reticências.
Dificilmente conseguiremos extrair muitos insights apenas com essa visualização, mas como o Power BI é uma ferramenta que gosta de ajudar nós analistas, ele disponibiliza a seguinte opção na aba de visualização dos dados:

Perfil da Coluna

Para quem já está familiarizado com a linguagem de programação Python para análise de dados, a library Pandas possui o método .describe muito semelhante com a ferramenta que eu irei apresentar.
Selecionando essa opção, este será o resultado:


Através dessa opção podemos extrair valores como:

  • Contagem
  • Valores nulos
  • Distintos e exclusivos
  • Valor mínimo
  • Valor máximo
  • Desvio padrão

Aquele gráfico da distribuição dos dados é ampliado mostrando os valores que possuem uma frequência maior e aqueles que possuem uma frequência menor.
Como por exemplo, nessa base de dados é possível analisar que a grande maioria das pessoas eram adultos e existiam uma quantidade menor de velhos e crianças.
Muito parecido com o .describe(), não é mesmo?!
Por padrão essa distribuição é baseada nas 1000 primeiras linhas do conjunto de dados. Neste caso eu estou trabalhando com apenas 891 linhas, então a ferramenta consegue atender a todas as entradas.
Se o conjunto de dados que você estiver utilizando possuir um número maior de entradas, é possível mudar a quantidade linhas, basta clicar em uma opção chamada "Criação do perfil de coluna com base nas primeiras 1000 linhas" que pode ser encontrada no canto inferior esquerdo:


E depois clicar em "Criação do perfil de coluna com base em todo o conjunto de dados":


Prontinho! Dessa maneira o Power Query irá se basear em todo o seu conjunto de dados!
Espero que essa dica relacionada ao Power Query ajude em sua produtividade em análises de dados e possa ser utilizada para extrair muitas informações relevantes.
